UX Portfolio

Won 1st Place Best Pitch at Garuda Hacks Hackathon by creating a UI/UX design of an online e-commerce platform (Tuku) for Indonesian crafts to revitalize and continue traditional practices

This is a branding style page that I created before actually making the mock-ups. I first designed a logo and a slogan, followed by coming up with a color scheme, and figuring out what typography I wanted to use on the website.

Registration Mockups

These are the baseline registration pages. The images above showcase the first welcome page when enterting the website to sign up, the second image showcases the basic account creation page, and the third image showcases the completion page.

These are the registration pages specifically for if the account user is a buyer. Additional information such as address and payment is requested, but the user can skip if they are not sure whether they want to input anything just yet.

These are the registration pages specifically for if the account user is a seller. Additional details such as more details about the shop (such as the name, artist information, and a shop description), address information, and bank account information is requested.
